# Design Direction

- Style: modern minimalist.
- Palette: light ecommerce-friendly UI with warm white backgrounds, slate text, emerald/teal primary actions, and restrained amber accents.
- Layout: clean spacing, clear hierarchy, dense enough for operations without feeling heavy.
- Arabic/RTL: Tajawal font, right-to-left layouts, concise Arabic interface copy.
- Avoid: dark default surfaces, heavy gradients, decorative blobs, avatar/photo systems, and marketing-style clutter.
